WP Maintenance Mode - New Version, Support Latest WP version 4.4,  Fixed all bugs. Quickly create a maintenance/coming-soon page. Use this plugin whenever your site is down for maintenance or undergoing development.


1. Download plugin zip file to your computer and extract in a folder
1. Upload `maintenance-mode` folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
